## Daemons
- Computer programs that run in background process is called _daemon_.

### Systemd
- System and service manager for linux

``/usr/lib/systemd/system`` - shows all available configuration of installed packages and service units

- service units ends with **.service**

### Systemctl

- Used to start/stop/restart services managed by **systemd**

Commands

``systemctl start name.service``

``systemctl stop name.service``

``systemctl restart name.service``

``systemctl status name.service``

``systemctl reload name.service``
