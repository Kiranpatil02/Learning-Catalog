## Process Management
``ps`` - shows information/list of processes

``ps aux`` - shows all currently running processes on the system.

``top`` - real time running processes

## Memory Managment

``free`` - displays memory usage of the system

``free -h`` - more readable output

``vmstat`` - display the memory usage along with additional information about IO and CPU usage.

## Disk space

``df``(disk free) - display the free and available space for each mounted file system

``du`` (disk usage) - display disk usage of files and directories on the system.
